# How to run the app.

## Deployment

To run this project run

```bash
  dfx start --clean
```

to deploy the project

```bash
  dfx deploy
```

intsall npm modules

```bash
  npm install

```

To stop all serivces

```bash
  dfx stop

```

Error stopping all services

```bash
  sudo kill -9 $(sudo lsof -t -i:4943

```
